# Moving Objects Hints and Tricks

## InfluxDB

```bash
brew install influxdb
```

## Git tagging
```bash
git tag new-tag old-tag
git tag -d old-tag
git push origin :refs/tags/old-tag
git push --tags
git pull --prune --tags
```

## How to build and run the Front End service

`npm install`

`npm run build`
